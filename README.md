# Object Counting using Python - OpenCV

This project demonstrates how to count the number of objects in an image using image processing techniques.

## Modules Used
- OpenCV
- NumPy
- Matplotlib

## Steps Performed
1. Load image
2. Convert to grayscale
3. Apply Gaussian blur
4. Detect edges using Canny
5. Dilate edges
6. Extract contours
7. Count objects
